# crypto-raw-key

crypto-raw-key allows you to import and export Edwards and Montgomery keys in a raw form.

### Installation

crypto-raw-key is available on npm.

```shell

# with npm:
npm install crypto-raw-key

# with yarn:
yarn add crypto-raw-key

```
